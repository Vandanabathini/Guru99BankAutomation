# Guru99 Banking Automation Framework

## Project Overview

This project is a Selenium Automation Framework developed for the Guru99 Banking Application using Java, Selenium WebDriver, TestNG, Maven, and Page Object Model (POM).

## Technologies Used

* Java
* Selenium WebDriver
* TestNG
* Maven
* Apache POI
* Git
* GitHub

## Framework Components

* BaseTest
* Page Classes
* Test Classes
* ConfigReader
* ExcelUtils
* WaitUtils
* ScreenshotUtil

## Automated Modules

* Login
* New Customer
* New Account
* Deposit
* Withdrawal
* Fund Transfer
* Balance Enquiry

## Framework Type

Hybrid Framework (POM + Data Driven Testing)
